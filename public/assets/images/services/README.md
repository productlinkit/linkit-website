# Service card mockups

Each card already ships with a built-in premium **SVG mockup** (`ds-1.svg … ds-8.svg`),
so the gallery looks finished out of the box.

## Want to swap in Higgsfield renders?

Generate one **portrait (3:4)** mockup per card and drop it here as `ds-1.webp … ds-8.webp`.
Cards layer the backgrounds as `url(.webp), url(.svg), gradient`, so **the moment a
`.webp` exists it automatically wins** — no code change, just refresh.

Recommended: ~1000×1333px (3:4), dark/premium tone so the white overlay text stays legible.
